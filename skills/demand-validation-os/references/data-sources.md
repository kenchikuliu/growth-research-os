# Data Sources

## Purpose

Use this file to keep the source roles narrow. The skill works best when each tool answers one question well instead of pretending every tool can answer everything.

## `gefei`

Use for judgment rules and page-thinking.

Typical outputs:

- user intent -> page type mapping
- “不要只做博客页”
- “不要只交关键词列表，要交页面清单”
- “找新词是长期有效策略，但要看生命周期和承接方式”

Use `gefei` when you need:

- a rule for whether a page type is correct
- a rule for whether a niche is worth doing
- a sanity check against common SEO self-deception

## `chuhai`

Use for practical operating paths.

Typical outputs:

- Similarweb landing-pages workflow
- Semrush top-pages workflow
- Google Trends usage notes
- keyword-finding SOP and demand-discovery framework

Use `chuhai` when you need:

- the exact evidence to pull from Similarweb
- the exact evidence to pull from Semrush
- a simple Trends interpretation path

## Google Trends

Use for trend shape, not absolute volume.

Primary questions:

- Is this a spike or a durable demand?
- Is wording migrating?
- Which countries matter first?
- Is there seasonality or event dependency?

Good evidence:

- 30-day shape
- 90-day shape
- 12-month shape
- 5-year shape
- Terms vs Topics
- rising related queries

## Similarweb

Use for result-side proof.

Primary questions:

- Which pages already got clicks?
- Which new pages are starting to move?
- What page types are showing up repeatedly?

Default path:

1. Keyword Research
2. Landing Pages
3. filter `点击量变化 -> 新点击量`
4. inspect page type
5. inspect Top Keywords

Do not treat Similarweb as a pure keyword ideation tool. Its value here is page-level proof.

## Semrush

Use for page-side and term-side decomposition.

Primary questions:

- Which pages actually carry the site?
- Which terms support those pages?
- Can the opportunity expand into a cluster?
- Are there supporting signals from ref domains or backlinks?

Default path:

1. Top Pages
2. Organic Positions
3. Keyword Gap
4. Backlinks / Ref Domains

Do not stop at total traffic. Top Pages is the minimum useful view.
